Seed Playframework 2.6 by nuboat
=================================
This project based on Play 2.7 + Scala. Follow Reactive (http://www.reactivemanifesto.org/).


Prerequsit
===========
- Intelli J 2016
- JDK 8
- brew install -y sbt


Test
=====
- sbt test


Test in IDe
====
- Change "Working Directory" from $MODULE_DIR$ to $PROJECT_DIR$


Standard Command
===========
- sbt clean | compile | test | dist
